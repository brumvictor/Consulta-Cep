# Consulta CEP
Projeto para consultar informações de endereços de todo o Brasil utilizando o webservice gratuito [ViaCEP](https://viacep.com.br/).  
O usuário pode inserir um CEP no formato `00000000` ou `00000-000` para obter dados como logradouro, bairro, cidade e estado.

## Funcionalidades
* Validação de CEP nos formatos aceitos (`00000000` ou `00000-000`).
* Consulta de informações detalhadas do endereço usando a API ViaCEP.
* Exibição dos resultados de forma simples e clara.

## Tecnologias
* HTML
* CSS
* JavaScript
